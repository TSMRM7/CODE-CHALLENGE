# JavaScript Toy Problems

This repository contains solutions to three JavaScript toy problems as described below:

#### Date, 2024/09/29

#### By *Ryan Wahinya

## Challenges

### 1. Student Grade Generator
A program that takes a student's marks as input and outputs the corresponding grade based on the provided scale.

- **File**: `student-grade-generator.js`
- **Input**: Marks between 0 and 100.
- **Output**: Grade (A, B, C, D, E).

### 2. Speed Detector
A program that checks the speed of a car and outputs the number of demerit points or suspends the license based on speed.

- **File**: `speed-detector.js`
- **Input**: Car speed.
- **Output**: Demerit points or "License suspended".

### 3. Net Salary Calculator
A program that calculates an individual's net salary based on basic salary and benefits. It accounts for PAYE (Tax), NHIF deductions, NSSF deductions, and the Housing Levy (2024 update).

- **File**: `net-salary-calculator.js`
- **Input**: Basic salary and benefits.
- **Output**: Net salary, including breakdown of tax, NHIF, NSSF, and housing levy deductions.
